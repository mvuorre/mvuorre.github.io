Void Mode Implementation Plan for vuorre.com

Project Overview

Create a hidden “void mode” for vuorre.com that transforms the academic website into an alternate reality experience. The mode focuses on creating weird-but-academic quantum versions of papers and research content while maintaining the site’s professional foundation.

Key Features

1. Void Mode Toggle

  • Extends existing dark mode with a third state (void)
  • Activates after 3 rapid clicks on dark mode toggle
  • Includes Konami code as alternate activation (↑↑↓↓←→←→ba)
  • Persists user preference in localStorage

2. Paper Quantum Transformations

  • Processes PDF content to create alternate reality versions
  • Maintains academic credibility while pushing weirdness
  • Includes preprints in transformation set
  • Preserves citation accuracy with quantum variations

3. Citation Network Visualizer

  • Visualizes papers as a neural network
  • Shows quantum probability citation metrics
  • Displays cross-dimensional research impact

4. Multiverse Research Timeline

  • Shows alternative research paths
  • Creates “what if” scenarios for research directions
  • Generates quantum probability publications

    // Core toggle functionality
const VOID_MODE_KEY = 'quarto-void-mode';
let voidClickCount = 0;
const VOID_TIMEOUT = 1000; // ms to reset click counter

function toggleVoidMode() {
  const html = document.querySelector('html');
  const isDark = html.classList.contains('dark');
  const isVoid = html.classList.contains('void');

  // Reset if already in void mode
  if (isVoid) {
    html.classList.remove('void');
    localStorage.setItem(VOID_MODE_KEY, 'false');
    voidClickCount = 0;
    return;
  }

  // Increment on dark mode toggle
  if (isDark) {
    voidClickCount++;

    // Enter void on third click
    if (voidClickCount >= 3) {
      html.classList.add('void');
      localStorage.setItem(VOID_MODE_KEY, 'true');
      voidClickCount = 0;
    }

    // Reset counter after timeout
    setTimeout(() => {
      voidClickCount = 0;
    }, VOID_TIMEOUT);
  }
}

// Add to existing dark mode listener
document.addEventListener('DOMContentLoaded', () => {
  const toggle = document.querySelector('.quarto-color-scheme-toggle');
  if (toggle) {
    toggle.addEventListener('click', toggleVoidMode);
  }

  // Initialize from localStorage
  if (localStorage.getItem(VOID_MODE_KEY) === 'true') {
    document.querySelector('html').classList.add('void');
  }
});

// Add Konami code support
const konamiCode = ['ArrowUp', 'ArrowUp', 'ArrowDown', 'ArrowDown', 'ArrowLeft', 'ArrowRight', 'ArrowLeft', 'ArrowRight', 'b', 'a'];
let konamiIndex = 0;

document.addEventListener('keydown', (e) => {
  if (e.key === konamiCode[konamiIndex]) {
    konamiIndex++;
    if (konamiIndex === konamiCode.length) {
      document.querySelector('html').classList.add('void');
      localStorage.setItem(VOID_MODE_KEY, 'true');
      konamiIndex = 0;
    }
  } else {
    konamiIndex = 0;
  }
});

Phase 2: PDF Processing and Alternate Reality Generation

  1. Create R script for PDF processing File: _scripts/void/process_pdfs.R

    library(pdftools)
library(tidytext)

# Process PDFs and extract content
process_pdf <- function(pdf_path) {
  # Extract text
  pdf_text <- pdf_text(pdf_path)

  # Extract sections (abstract, methods, results, etc.)
  # This is a simplified version - will need customization
  sections <- extract_sections(pdf_text)

  # Return structured content
  return(sections)
}

# Extract key sections from PDF text
extract_sections <- function(pdf_text) {
  # Combine all pages
  full_text <- paste(pdf_text, collapse = " ")

  # Extract basic sections
  # This will need refinement based on your papers' structure
  abstract <- extract_between(full_text, "Abstract", "Introduction")
  introduction <- extract_between(full_text, "Introduction", "Method")
  methods <- extract_between(full_text, "Method", "Results")
  results <- extract_between(full_text, "Results", "Discussion")
  discussion <- extract_between(full_text, "Discussion", "References")

  # Return as list
  list(
    abstract = abstract,
    introduction = introduction,
    methods = methods,
    results = results,
    discussion = discussion
  )
}

# Helper function to extract text between section headers
extract_between <- function(text, start, end) {
  pattern <- paste0(start, ".*?", end)
  matches <- regexpr(pattern, text, perl = TRUE)
  if (matches > 0) {
    result <- regmatches(text, matches)
    # Remove the end pattern
    result <- gsub(paste0("\\s*", end, ".*"), "", result)
    # Remove the start pattern
    result <- gsub(paste0(start, "\\s*"), "", result)
    return(result)
  }
  return("")
}

  2. Create R script for quantum text generation File: _scripts/void/quantum_generator.R

    library(stringr)

# Generate quantum version of paper content
generate_quantum_paper <- function(sections) {
  # Transform each section
  quantum_sections <- list(
    abstract = quantum_transform(sections$abstract, "abstract"),
    introduction = quantum_transform(sections$introduction, "introduction"),
    methods = quantum_transform(sections$methods, "methods"),
    results = quantum_transform(sections$results, "results"),
    discussion = quantum_transform(sections$discussion, "discussion")
  )

  return(quantum_sections)
}

# Transform text into quantum version
quantum_transform <- function(text, section_type) {
  # Replace common phrases with quantum alternatives
  # These patterns will need adjustment based on your papers
  quantum_replacements <- list(
    # Abstract transformations
    c("We conducted", "We conducted across multiple dimensions"),
    c("significant effect", "quantum-entangled effect"),
    c("data show", "data rippling across realities show"),

    # Methods transformations
    c("participants", "participants (existing simultaneously in parallel dimensions)"),
    c("were recruited", "were recruited across the multiverse"),
    c("randomly assigned", "quantum-randomly assigned"),

    # Results transformations
    c("analysis revealed", "trans-dimensional analysis revealed"),
    c("statistically significant", "quantum-probabilistically significant"),
    c("effect size", "effect size (collapsed across all possible realities)"),

    # Statistical transformations
    c("p < 0.05", "p < 0.05 in this particular slice of the multiverse"),
    c("confidence interval", "quantum probability corridor"),
    c("standard deviation", "quantum uncertainty parameter")
  )

  # Apply transformations
  transformed_text <- text
  for (replacement in quantum_replacements) {
    transformed_text <- str_replace_all(transformed_text,
                                        fixed(replacement[1]),
                                        replacement[2])
  }

  # Add section-specific quantum elements
  if (section_type == "abstract") {
    transformed_text <- paste0(transformed_text,
                              " The findings exist simultaneously across multiple research timelines.")
  }

  if (section_type == "methods") {
    transformed_text <- paste0("In this dimension's iteration of the methodology, ", transformed_text)
  }

  if (section_type == "results") {
    transformed_text <- str_replace(transformed_text,
                                   "\\.",
                                   ". These results may differ in parallel research timelines.")
  }

  return(transformed_text)
}

# Get all PDFs
pdf_dir <- "papers/"
pdf_files <- list.files(pdf_dir, pattern = "\\.pdf$", full.names = TRUE)

# Process each PDF
for (pdf_file in pdf_files) {
  # Extract filename
  filename <- basename(pdf_file)
  paper_id <- tools::file_path_sans_ext(filename)

  # Process PDF
  sections <- process_pdf(pdf_file)

  # Generate quantum version
  quantum_sections <- generate_quantum_paper(sections)

  # Save to _void directory
  output_dir <- file.path("_void/papers", paper_id)
  dir.create(output_dir, recursive = TRUE, showWarnings = FALSE)

  # Write quantum version
  write_json(quantum_sections, file.path(output_dir, "quantum.json"))
}

document.addEventListener('DOMContentLoaded', () => {
  if (!document.querySelector('html').classList.contains('void')) {
    window.location.href = "/publications.html";
    return;
  }

  // Get standard publications from page
  const standardPubs = Array.from(document.querySelectorAll('.quarto-grid-item'));

  // Create void versions
  const voidList = document.querySelector('.void-publication-list');
  standardPubs.forEach(pub => {
    // Clone the publication
    const voidPub = pub.cloneNode(true);
    voidPub.classList.add('void-publication');

    // Get the title
    const title = voidPub.querySelector('h5').textContent;

    // Transform title
    const quantumTitle = getQuantumTitle(title);
    voidPub.querySelector('h5').textContent = quantumTitle;

    // Add to the list
    voidList.appendChild(voidPub);
  });
});

// Generate quantum paper title
function getQuantumTitle(title) {
  const quantumPrefixes = [
    "Quantum Entanglement of ",
    "Trans-dimensional Analysis of ",
    "Multiverse Implications for ",
    "Probability Wave Collapse in ",
    "Quantum Superposition of "
  ];

  const quantumSuffixes = [
    " Across Parallel Dimensions",
    " in the Academic Multiverse",
    " and its Quantum Eigenstates",
    " Beyond Conventional Reality",
    " Through the Void"
  ];

  // Add prefix (70% chance)
  let quantumTitle = title;
  if (Math.random() < 0.7) {
    const prefix = quantumPrefixes[Math.floor(Math.random() * quantumPrefixes.length)];
    quantumTitle = prefix + quantumTitle;
  }

  // Add suffix (50% chance)
  if (Math.random() < 0.5) {
    const suffix = quantumSuffixes[Math.floor(Math.random() * quantumSuffixes.length)];
    quantumTitle = quantumTitle + suffix;
  }

  return quantumTitle;
}
</script>

<script>
// Only execute in void mode
document.addEventListener('DOMContentLoaded', () => {
  if (!document.querySelector('html').classList.contains('void')) {
    window.location.href = "/publications.html";
    return;
  }

  // Simple force-directed graph of papers
  const papers = [
    // These would ideally be generated from actual paper data
    { id: "paper1", title: "Quantum Memory", year: 2020 },
    { id: "paper2", title: "Multiverse Psychology", year: 2021 },
    { id: "paper3", title: "Trans-dimensional Methods", year: 2022 },
    { id: "paper4", title: "Reality Collapse", year: 2023 }
  ];

  // Create random links between papers
  const links = [];
  for (let i = 0; i < papers.length; i++) {
    for (let j = i + 1; j < papers.length; j++) {
      if (Math.random() < 0.7) {
        links.push({
          source: papers[i].id,
          target: papers[j].id,
          strength: Math.random()
        });
      }
    }
  }

  // Create visualization
  const width = 800;
  const height = 600;

  const svg = d3.select("#citation-network")
    .append("svg")
    .attr("width", width)
    .attr("height", height);

  // Create force simulation
  const simulation = d3.forceSimulation(papers)
    .force("link", d3.forceLink(links).id(d => d.id))
    .force("charge", d3.forceManyBody().strength(-200))
    .force("center", d3.forceCenter(width / 2, height / 2));

  // Draw links
  const link = svg.append("g")
    .selectAll("line")
    .data(links)
    .enter().append("line")
    .attr("stroke", "#b300ff")
    .attr("stroke-opacity", d => d.strength)
    .attr("stroke-width", d => d.strength * 5);

  // Draw nodes
  const node = svg.append("g")
    .selectAll("circle")
    .data(papers)
    .enter().append("circle")
    .attr("r", 10)
    .attr("fill", "#6e0095")
    .call(d3.drag()
      .on("start", dragstarted)
      .on("drag", dragged)
      .on("end", dragended));

  // Add labels
  const label = svg.append("g")
    .selectAll("text")
    .data(papers)
    .enter().append("text")
    .text(d => d.title)
    .attr("font-size", 12)
    .attr("dx", 15)
    .attr("dy", 4)
    .attr("fill", "#e0c8ff");

  // Update positions
  simulation.on("tick", () => {
    link
      .attr("x1", d => d.source.x)
      .attr("y1", d => d.source.y)
      .attr("x2", d => d.target.x)
      .attr("y2", d => d.target.y);

    node
      .attr("cx", d => d.x)
      .attr("cy", d => d.y);

    label
      .attr("x", d => d.x)
      .attr("y", d => d.y);
  });

  // Drag functions
  function dragstarted(event, d) {
    if (!event.active) simulation.alphaTarget(0.3).restart();
    d.fx = d.x;
    d.fy = d.y;
  }

  function dragged(event, d) {
    d.fx = event.x;
    d.fy = event.y;
  }

  function dragended(event, d) {
    if (!event.active) simulation.alphaTarget(0);
    d.fx = null;
    d.fy = null;
  }
});
</script>

Phase 4: Navigation Enhancements

  1. Add void-mode navigation links to standard navigation Add to void-mode.js:

    // Add void navigation
function addVoidNavigation() {
  const html = document.querySelector('html');
  if (!html.classList.contains('void')) return;

  const navbar = document.querySelector('.navbar-nav');
  if (!navbar) return;

  // Check if void nav already exists
  if (document.querySelector('.void-nav-link')) return;

  // Create void navigation items
  const voidLinks = [
    { href: '/_void/publications.html', text: 'Quantum Papers' },
    { href: '/_void/network.html', text: 'Citation Cosmos' }
  ];

  voidLinks.forEach(link => {
    const li = document.createElement('li');
    li.className = 'nav-item void-nav-link';

    const a = document.createElement('a');
    a.href = link.href;
    a.className = 'nav-link';
    a.textContent = link.text;
    a.style.color = 'var(--void-accent)';

    li.appendChild(a);
    navbar.appendChild(li);
  });
}

// Initialize void navigation
document.addEventListener('DOMContentLoaded', () => {
  // Check initial state
  if (localStorage.getItem(VOID_MODE_KEY) === 'true') {
    addVoidNavigation();
  }

  // Observer for void mode changes
  const observer = new MutationObserver((mutations) => {
    mutations.forEach((mutation) => {
      if (mutation.attributeName === 'class') {
        const isVoid = document.querySelector('html').classList.contains('void');
        if (isVoid) {
          addVoidNavigation();
        } else {
          document.querySelectorAll('.void-nav-link').forEach(el => el.remove());
        }
      }
    });
  });

  observer.observe(document.querySelector('html'), { attributes: true });
});

  4. Maintain proper citations

Integration with Existing Site

The void mode is designed to be non-intrusive: 1. Standard site functions normally without void mode active 2. Void mode is activated explicitly by user 3. Blog section remains untouched 4. Navigation supports both modes 5. Content maintains academic credibility

This implementation provides a parallel quantum reality for your academic work while keeping the standard professional interface accessible and functional.
